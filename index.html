Graduation Prognosis

Graduation Prognosis is a Machine Learning based prediction project which has its frontend in React and the backend is rendered using Flask. The model used in the project predicts the chances of a student getting admission for Masters in colleges around the US. It takes into consideration various parameters like GRE score, TOEFL score, CGPA, Research projects etc.

BookKeeper NITJ

Bookkeepernitj is a web based application which helps students save money on books. Students can Signup with Google and upload details of books available with them. Users in need of these books can search for the availability on the web app and can contact the book lenders directly. It uses Bootstrap for Front End and Express for Back End. Other libraries used include Passport for authentication, Mongodb, mongoose for managing the database along with various other dependencies.
